# Oral Cancer Prediction using Machine Learning

## Overview
Early detection of oral cancer can significantly improve survival rates. This project builds a machine learning model to predict the likelihood of oral cancer based on medical and lifestyle features.

The goal is to demonstrate how data science can assist healthcare professionals in early diagnosis.

---

## Project Highlights
- Data preprocessing and feature engineering
- Exploratory data analysis
- Machine learning model training
- Model evaluation using multiple metrics
- Prediction pipeline

---

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook
